# M295_LB-B
## Autor: Heiko Ho
## Zweck: Funktionstüchtiges Backend für die Leistungsbeurteilung B im Modul 295.

## Technologien

- **Sprache**: *JavaScript*
- **Runtime**: *Node.js*
- **Framework**: *Express.js*, *Express-Session*

## Installation

1. Repository klonen
```bash
git clone https://github.com/bourb0n1412/M295_LB-B.git
```
2. Dependencies installieren
```bash
cd M295_LB-B
npm install
```
3. Server starten
```bash
cd src
node server.js
```
